# Asset Requests

Use the existing `public/assets/sandbox-bg.png` as the visual anchor. Keep everything in the same cozy pixel-cartoon garden style: chunky silhouettes, clean readable shapes, warm daylight, hand-painted pixel texture, no glossy 3D render, no generic AI fantasy lighting.

## Highest Priority

### 1. Creature sprite sheet

**File target:** `public/assets/creature-sprites.png`

Generate a transparent PNG sprite sheet with 20 small NFT garden creatures. Each creature should fit a square cell, read clearly at 64-96 px, and feel like it belongs on the existing garden platforms.

Prompt direction:

```text
Transparent PNG sprite sheet, 20 unique tiny garden NFT creatures arranged in a clean 5 by 4 grid, cozy pixel-cartoon style matching a hand-painted pixel garden sandbox. Chunky cute silhouettes, expressive faces, little leaves, sprouts, petals, mushrooms, crystal buds, soft monster plant forms, readable at 64px. Warm pastel colors with purple, pink, mint, yellow, teal accents. Crisp pixel edges, subtle hand-painted shading, no background, no text, no UI, no glossy 3D, no photorealism, no blur, no AI-looking overdetail.
```
